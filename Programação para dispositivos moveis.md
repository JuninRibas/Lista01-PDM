Programação para dispositivos moveis
1) O é a classe R em um projeto android?
R: A classe R no Android é um arquivo gerado automaticamente pelo sistema de compilação do Android Studio, e armazena referências a recursos do aplicativo.
2) O que Activity no Android, para que serve?
R: O Activity é o componente responsável pelo visual, possibilitando a visualização das telas e componentes e a interação entre eles.
3) O que é o arquivo Manifesto em um projeto Android e quais informações podem ser declaradas nele.
R: O arquivo Manifesto é um arquivo XML obrigatório em um projeto Android que fornece informações importantes sobre o aplicativo. Geralmente localizado na raiz do diretório do projeto e com o nome de AndroidManifest.xml. Informações importantes são declaradas como Nome do pacote do aplicativo, Versão do aplicativo, Permissões Componentes do aplicativo entre outras funções.
4)O que são os níveis de API de uma aplicação Android e o que implica uma aplicação utilizar uma determinada API.
R: Os níveis de API de uma aplicação Android referem-se às versões da API do Android que a aplicação suporta. Quando se tem um maior nível de API, abre-se u leque de recursos e funcionalidades, mas o aplicativo só poderá ser executado em dispositivos com essa versão do Android ou superior.
5) O que é constraintLayout no android? Para que serve, cite ao menos dois exemplos de uso com posicionamento de um componente qualquer.
R: ConstraintLayout é um tipo de layout que permite definir as posições e restrições dos componentes com base em suas relações com outros componentes.
 Dois exemplos de uso do ConstraintLayout com posicionamento de um componente qualquer são: Alinhamento vertical e horizontal e  Posicionamento relativo.
